## üé® Customization Guide

### Changing Colors
Modify CSS variables in `:root` selector:
```css
:root {
    --primary: #00ffcc;    /* Main accent color */
    --secondary: #ff00ff;  /* Secondary accent */
    --dark: #0a0a1a;       /* Background color */
    --light: #f0f0f0;      /* Text color */
}
```

### Updating Content
1. **Personal Information**: Replace name, bio, and contact details
2. **Projects**: Update project images, titles, and descriptions
3. **Skills**: Modify skill bars and percentages
4. **Social Links**: Update social media profile URLs

### Adding New Sections
```html
<!-- Add new section following the existing pattern -->
<section id="new-section">
    <div class="section-title">
        <h2>Section Title</h2>
        <p>Section description</p>
    </div>
    <div class="section-content">
        <!-- Your content here -->
    </div>
</section>
```
